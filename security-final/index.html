Invite Code

I first go to the website and open the Developer Console tools and refresh to view all of the network connections. I inspect each and the contents that are transferred but don’t see any useful information. I also inspect all of the files being loaded and content on the page.

Based on the lesson from Week 7, I try hacking in through modifying the form but only get this message:

I do see a javascript file being loaded called ’/js/inviteapi.min.js’ which looks like a minified javascript file. I execute this script in jsbin.com but change “return p” to “console.log(p)” to see what it says: "function verifyInviteCode(code){var formData={\"code\":code};$.ajax({type:\"POST\",dataType:\"json\",data:formData,url:'/api/invite/verify',success:function(response){console.log(response)},error:function(response){console.log(response)}})}function makeInviteCode(){$.ajax({type:\"POST\",dataType:\"json\",url:'/api/invite/how/to/generate',success:function(response){console.log(response)},error:function(response){console.log(response)}})}"

Here I see maybe I should do a CURL Post request to /api/invite/how/to/generate!

This gives me {"success":1,"data":{"data":"SW4gb3JkZXIgdG8gZ2VuZXJhdGUgdGhlIGludml0ZSBjb2RlLCBtYWtlIGEgUE9TVCByZXF1ZXN0IHRvIC9hcGkvaW52aXRlL2dlbmVyYXRl","enctype":"BASE64"},"0":200}.

I try that code but it doesn’t work, so I see it also says “BASE64” so I try decoding it using https://www.base64decode.org and find out it decodes to “In order to generate the invite code, make a POST request to /api/invite/generate”.

I do a CURL POST to /api/invite/generate and get the response {"success":1,"data":{"code":"VFZJVk4tQVdFWEctWlpWUVMtWUdSSkgtR0FDV0c=","format":"encoded"},"0":200}. I try that code, but that doesn’t work, so I also BASE64 decode that to get TVIVN-AWEXG-ZZVQS-YGRJH-GACWG. That works!

Challenge 2 - CRYPTO - SICK TEACHER - 20 points

This looks like a classic crypto challenge. I first think that one of the words might be “teacher” but the pattern doesn’t fit any. I notice that “IKW” repeats and I think it could be “THE” because there is also “IKWYW” which could be “THERE”. When I put in “THERE”, I notice the next word could be “ARE”. I fill in all of those letters. Next I figure that C = O because I have T? which is probably TO. Next I notice OR?ER so I think S must be “D”. “HMS” so far translates to “A?D” so I think “M” = “N”. “B” must equal “I” since we already know what “A” is and there’s no other single letter word. Next I guess “J” = “S” because of “IKBJ” being “THI?” and “BJ” being “I?“. From there, I keep going and guessing letters until the entire sentence becomes clear. (Although I regret wasting so much time filling in the entire thing since the flag is at the very end — this does occur to me that it may be at the end, but I keep going since I do want to solve the entire cipher.)
